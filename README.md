# Canvas 
A simplified imitation of Canvas, supporting three basic features, adding item from a sheet, moving the added item on the canvas and the snap feature that we have for items when theyâ€™re being moved around on the canvas.

# Features
* Consume an API to fetch images
* Add images to canvas
* Select and deselect by tapping out 
* Move and scale objects
* Snap to edge of other images and canvas

# Tech stack
* SwiftUI
* MVVM pattern
